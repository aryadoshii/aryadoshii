## 💎 Featured Projects

### 🤖 AI-Powered Porosity Detection | *Production System @ Aaravali Tech*
End-to-end CV pipeline for automotive manufacturing QC — **91.10% accuracy**, **3.76x model compression** via INT8 quantization, **78% reduction** in manual inspection time. Proved architectural simplicity beats complexity with noisy labels (r = -0.96).

`Python` `PyTorch` `OpenCV` `TensorRT` `FastAPI` `Docker`

---

### 💰 QuantEdge: Systematic Equity Trading Platform
Market-neutral long-short strategy — **1.19 Sharpe**, **2.03% max drawdown**, 32+ quantitative factors. Key insight: shifted from return prediction to stock ranking, transforming an unprofitable model into a consistently profitable strategy.

`Python` `scikit-learn` `XGBoost` `yfinance` `Streamlit` `Plotly`

---

### 🧠 MedSynGAN: Federated Medical Image Generation | 🏆 Best Paper @ ICCVR 2025
Privacy-preserving federated GAN framework generating synthetic chest X-rays without raw data leaving hospitals — **91% diagnostic accuracy**, **1024x1024 resolution**. Enables medical AI research while maintaining HIPAA/GDPR compliance.

`Python` `TensorFlow` `Flower Framework` `DICOM` `Federated Learning` · [IEEE Xplore →](https://ieeexplore.ieee.org/document/10817878)

---

### 📊 Statistical Arbitrage Trading System | *Gemscap Assessment*
Real-time crypto pairs trading with WebSocket feeds — cointegration testing, z-score signals, VaR-based risk management. Built with TimescaleDB + Redis for high-performance time-series analytics.

`Python` `Async Programming` `TimescaleDB` `Redis` `Streamlit` `Binance API`
